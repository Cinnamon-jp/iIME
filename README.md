utoLangIME (仮称)

ユーザーに「入力モードの切り替え」を意識させない、完全自動言語判別型の次世代自作IME（インプットメソッド）です。

##  概要

従来のIMEでは、日本語（かな）と英語（アルファベット）を切り替えるために、毎回 `半角/全角` キーや `Caps` キー、`Cmd + Space` などを入力する必要がありました。

**AutoLangIME** は、ユーザーからの入力を常にアルファベットとして受け付け、文脈や入力パターンから**IME側が自動で言語を判断・変換**します。これにより、モード切り替えのパラドックスから解放され、シームレスでストレスフリーなタイピング体験を実現します。

## ✨ 主な特徴

* **常時アルファベット受付:** 入力モードは1つだけ。常に英語を入力する感覚でタイピングを始められます。
* **自動言語判別 (Auto-Language Detection):** 入力された文字列（ローマ字パターン、英単語のスペルなど）をリアルタイムに解析し、日本語か英語（あるいは他言語）かを自動で判定します。
* **シームレスな変換:** 
  * `nihongo` と打てば、自動で「日本語」と変換候補が出ます。
    * `welcome` と打てば、そのまま「welcome」として確定、あるいはスムーズに英語の予測補完が行われます。
    * **タイピングストレスの軽減:** 「あ、英語モードのまま日本語打っちゃった…」というミスが根本的に発生しなくなります。

## 🎯 ターゲット・ユースケース

* プログラミング中にコード（英語）とコメント・ドキュメント（日本語）を頻繁に行き来する開発者
* チャットやメールで日本語と英語を混ぜてスピード感持って執筆するビジネスパーソン
* 入力モードの切り替え忘れに毎日ストレスを感じているすべての人
